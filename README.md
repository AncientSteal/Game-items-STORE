# Магазин предметов из онлайн игры Genshin Impact

Данный сайт представляет собой каталог товаров. Позволяет пользователям добавлять и удалять товары в корзину, переходить к окну оформления заказа, а также предоставляет доступ к тематическим ресурсам.

---

## Структура проекта

- **shop** — основная папка.
   - **my_app** — главная папка с файлами моделей (models) и контролером (views)
      - **templates** — папка с шаблонами html
   - **shop** — папка с файлами настроек (settings) и конфигурации маршрутов (urls)
   - **static**
      - **css** — стили оформления сайта.
      - **images** — изображения.
      - **js** — файлы с кодом java script.

---

## Основные особенности

- Заголовок и стилизация страницы для удобного визуального восприятия.
- Каталог товаров с возможностью просмотра карточек преметов.
- Сайд-меню с полезными ссылками.
- Отдельная страница с корзиной выбранных товаров.
- Возможность удаления/добавления товаров в корзину.

---

## Как запустить проект

1. Открыть проект (папку shop) в любом удобном редакторе кода поддерживающим Python (например, PyCharm).
2. Провести конфигурацию с postgreSQL на вашем компьютере, в файле settings в разделе DATABASES выставить свои данные для работы с базой данных.
3. Установить необходимые расширения django и psycopg через терминал
4. Провести миграцию файлов
5. Запустить сервер с помощью команды: python manage.py runserver
